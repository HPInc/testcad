# testcad
Test design toolkit application

This project has been developed using the Qt framework. (https://www.qt.io/)

You can download a ready to be run testCad copy for each of the supported platforms
each copy contains the required dynamically linked Qt libraries.

Alternatively, if you plan to modify/compile testCad you are advised to:

1. Download and install Qt. (https://www.qt.io/download)
2. Ensure your installation references a valid Compiler and Debugger.
3. Download testCad source code.
4. Compile use and contribute !
